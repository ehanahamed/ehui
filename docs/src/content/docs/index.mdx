## Get Started

<Steps>

1. Link or download EhUI's main css file
   ```html
    <link rel="stylesheet" href="https://ehan.dev/ehui/css/ehui.min.css">
   ```

2. Add a theme css file
   ```html {2}
   <link rel="stylesheet" href="https://ehan.dev/ehui/css/ehui.min.css">
   <link rel="stylesheet" href="https://ehan.dev/ehui/themes/auto-light-dark.min.css">
   ```

3. You're all set

</Steps>
